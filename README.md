# Analyse des facteurs de risque des AVC

## Contexte

Les accidents vasculaires cérébraux (AVC) représentent l'une des principales causes de mortalité et de handicap dans le monde.

L'objectif de ce projet est d'explorer les caractéristiques médicales et démographiques des patients afin d'identifier les facteurs associés au risque d'AVC et de restituer les résultats à travers un dashboard Power BI.

## Objectifs

- Explorer et comprendre les données patients
- Nettoyer et préparer les données
- Identifier les facteurs de risque associés aux AVC
- Réaliser des analyses statistiques descriptives
- Construire des visualisations interactives
- Développer un dashboard Power BI

## Technologies

- Python
- Pandas
- SQL
- Power BI
- Git
- GitHub
